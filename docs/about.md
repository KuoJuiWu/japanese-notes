# 關於這個筆記系統

這個網站的筆記由一個 Telegram bot 自動生成。每當你在 Telegram 輸入一個日文句子，bot 會自動分析、查詢，並將結果儲存為結構化的 Markdown 筆記。

---

## 筆記欄位說明

### 原句 (Original)
你輸入的原始日文字串，原封不動保存。

### 分詞解析 (Morphological Analysis)
使用 **fugashi + UniDic** 將句子拆成詞素，每個詞素標注：

| 欄位 | 說明 |
|---|---|
| 表面形 | 文字中實際出現的字串 |
| 讀音 | 平假名讀音 |
| 詞性 | 名詞／動詞／助動詞／助詞 等 |
| 原形 | 辭書形，例如「食べ」→「食べる」 |

### 意思 (Meaning)
使用 **JMdict 離線字典**自動查詢，依詞性篩選最相關的意思。只顯示名詞、動詞、形容詞、副詞，助詞和助動詞的說明由下方文法區塊處理。

### 文法 (Grammar)
**Step 2**：辨識句子中的助動詞和複合語尾，例如：

| 語尾 | 說明 |
|---|---|
| 〜ていない | 進行否定（還沒在做） |
| 〜てしまう | 完了／遺憾 |
| ない | 否定 |
| た | 過去／完了 |

若遇到未收錄的助動詞，會顯示 ⚠️ 未收錄 提示。

### 文法模板 (Grammar Pattern)
**Step 3**：比對句子中的高頻文法結構，例如：

| 模板 | 說明 |
|---|---|
| Vたことがある | 曾經～過（經驗） |
| Vなければならない | 必須～（義務） |
| VことができるV | 能夠～（可能） |
| Vてから | 做完～之後（順序） |

目前收錄約 50 個模板，涵蓋 N5 至 N1 常見句型。

### 例句 (Example)
使用 **Massif 語料庫** API 自動查詢，優先搜尋包含原句或關鍵詞的真實例句。若查詢失敗會顯示警告。

### 出典 (Source)
例句來源。自動查詢標注 `Massif`，手動輸入標注 `手動入力`。

### 参考 (Reference)
Kotobank 和 Weblio 的查詢連結，方便進一步確認。
