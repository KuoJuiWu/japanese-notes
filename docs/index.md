# 🇯🇵 My Japanese Notes

A personal Japanese learning workspace with two systems:

## 📓 Vocabulary Notes
Personal notes generated by my Telegram bot. Send Japanese text, the bot analyzes it and saves a structured note.

Each note includes:
- 分詞解析 (word breakdown)
- 意思 (meaning)
- 例句 (example sentence)
- 出典 (source)
- 参考 (references)
