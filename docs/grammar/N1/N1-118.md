# ~ならいざしらず

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | A is understandable, but B is not |

## Example

新人ならいざしらず、ベテランがこんなミスをするなんて信じられない。

> It might be understandable for a rookie, but I can't believe a veteran would make such a mistake.
