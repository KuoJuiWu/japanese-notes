# ~といったらない / といったらありゃしない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | incredibly / nothing is more... |

## Example

スマホを忘れただけなのに、一日中不便といったらなかった。

> Even though I just forgot my smartphone, it was incredibly inconvenient all day.
