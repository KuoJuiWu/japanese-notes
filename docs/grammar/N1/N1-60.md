# ~たら〜ところだ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | would have been / was about to... |

## Example

警察につかまるなんて、父が生きていたら殴られてるところだ。

> Getting caught by the police—if my father were alive, I would have been beaten for this.
