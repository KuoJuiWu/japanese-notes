# ~に忍びない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | cannot bring oneself to / bear to... |

## Example

今までもらったプレゼントを全部捨てるに忍びないから、売ってしまおうか。

> I can't bear to throw away all the presents I've received so far, so maybe I should sell them?
