# 〜んがために

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in order to... |

## Example

金メダルを取らんがために、ライバルより1分でも長く練習する。

> In order to win a gold medal, I practice even one minute longer than my rivals.
