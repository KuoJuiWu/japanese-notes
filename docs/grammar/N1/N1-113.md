# ~ないでもない / ないものでもない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | it is not that... not... |

## Example

あなたの気持ちはわからないでもないんだけど、まずは謝るべきだよ。

> It's not that I don't understand how you feel, but you should apologize first.
