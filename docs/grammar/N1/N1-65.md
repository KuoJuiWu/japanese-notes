# ~つ~つ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | doing... and doing... |

## Example

レースは抜きつ抜かれつ、最後まで結果がわからなかった。

> In the race, they kept passing and being passed, and the result was unknown until the very end.
