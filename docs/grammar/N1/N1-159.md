# ~はおろか

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | let alone / not to mention... |

## Example

今のままでは大学はおろか、短大に入るのさえ難しいですよ。

> As it is now, it will be difficult to even get into a junior college, let alone a university.
