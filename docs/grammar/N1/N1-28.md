# ~ことこの上ない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | nothing more... than... |

## Example

第一志望の大学に合格し、うれしいことこの上ない。

> I passed my first-choice university, and there is nothing I am happier about.
