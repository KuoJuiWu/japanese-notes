# ~そばから

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | as soon as (repeatedly)... |

## Example

漢字が苦手で、覚えたそばから忘れてしまう。

> I'm bad at kanji; I forget them as soon as I memorize them.
