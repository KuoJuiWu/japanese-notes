# ~に照らして

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in light of / according to... |

## Example

常識に照らして考えれば、彼の行動は大人のものとは思えない。

> In light of common sense, his actions cannot be considered those of an adult.
