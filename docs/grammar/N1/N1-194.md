# ~を限りに

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | as the last / starting from... |

## Example

今日を限りに、もうお酒は一滴も飲まないことにする。

> With today as the last, I've decided not to drink even a single drop of alcohol.
