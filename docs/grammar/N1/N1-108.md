# ~と見るや

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | as soon as one sees/realizes... |

## Example

警察と見るや、若者グループはバラバラに逃げ出した。

> As soon as they saw the police, the group of youths scattered and ran away.
