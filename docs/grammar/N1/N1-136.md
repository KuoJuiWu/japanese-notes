# ~にかまけて

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | being too busy with / engrossed in... |

## Example

仕事にかまけて、子供と遊ぶ時間も取らなかったことを後悔している。

> I regret that I was so engrossed in work that I didn't take time to play with my children.
