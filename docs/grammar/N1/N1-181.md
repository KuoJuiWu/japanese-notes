# ~ものと思われる / ものと見られる

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | it is thought that / it seems that... |

## Example

まだコーヒーが温かいので、さっきまでここにいたものと思われます。

> The coffee is still warm, so it is thought that they were here until just a moment ago.
