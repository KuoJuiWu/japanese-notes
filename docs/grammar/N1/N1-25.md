# こそあれ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | although there is... |

## Example

小さなミスこそあれ、ほとんど完璧な仕事をするのは彼女しかいない。

> Although she makes small mistakes, she is the only one who does almost perfect work.
