# ~そびれる

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | miss the chance to... |

## Example

ガソリンを入れそびれて、なくならないかヒヤヒヤしている。

> I missed the chance to get gas, and now I'm nervously worrying it might run out.
