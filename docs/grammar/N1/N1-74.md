# ~ではあるまいし / でもあるまいし

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | it's not as if... |

## Example

子供ではあるまいし、自分のことは自分でしなさい。

> It's not like you're a child, so take care of your own things.
