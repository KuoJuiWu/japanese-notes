# ~では済まない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | can't get away with just... |

## Example

部下が問題を起こしたら、上司もただでは済まないよ。

> If a subordinate causes a problem, the boss won't get away unscathed either.
