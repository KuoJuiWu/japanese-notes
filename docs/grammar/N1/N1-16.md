# ~からある / からする/ からの

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | as much as / as many as... |

## Example

婚約に際し、彼から300万円からする指輪が贈られた。

> Upon our engagement, he gave me a ring worth as much as 3 million yen.
