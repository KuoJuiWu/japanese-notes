# ~さ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | (strong assertion/certainty) |

## Example

気にすることないさ。明日には課長の機嫌も治っているさ。

> There's nothing to worry about. The section chief's mood will surely be better tomorrow.
