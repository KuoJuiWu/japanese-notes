# ~ときたら

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | when it comes to... |

## Example

うちの娘ときたら、勉強もせずゲームばかりしてるんですよ。

> When it comes to my daughter, she doesn't study and just plays games all the time.
