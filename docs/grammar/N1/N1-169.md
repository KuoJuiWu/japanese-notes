# ~べくして

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | inevitably / as expected... |

## Example

対策もなにもなかったため、この事件は起こるべくして起こったと言える。

> Because there were no countermeasures, it can be said that this incident was bound to happen.
