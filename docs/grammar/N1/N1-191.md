# ~をおいて他にない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | none other than / only... |

## Example

哲学について学びたいなら、中田教授のいるこの大学をおいて他にない。

> If you want to study philosophy, there is no other place than this university where Professor Nakata is.
