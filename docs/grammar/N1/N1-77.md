# ~てみせる

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | I'll show you that I can... |

## Example

私は超能力者です。あなたの心の中を私が読んでみせましょう。

> I am a psychic. I will show you that I can read your mind.
