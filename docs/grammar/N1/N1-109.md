# ~とも〜とも

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | unable to say whether A or B |

## Example

いいとも悪いとも言えないよ、あなたの選択だからね。

> I can't say whether it's good or bad, because it's your choice.
