# ~にひきかえ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in contrast to... |

## Example

よくできる兄にひきかえ、弟の私は何をやってもうまくいかない。

> In contrast to my capable older brother, I, the younger brother, fail at whatever I do.
