# ~に難くない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | not difficult to... |

## Example

弁護士になる道のりが厳しいことは想像に難くない。

> It is not difficult to imagine that the path to becoming a lawyer is tough.
