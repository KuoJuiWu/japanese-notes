# ~もさることながら

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | not only... but also... |

## Example

その漫画は絵もさることながら、ストーリーもすばらしくて目が離せない。

> Not only is the art in that manga great, but the story is also wonderful and keeps you hooked.
