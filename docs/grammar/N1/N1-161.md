# ~はさておき

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | setting aside... |

## Example

年齢はさておき、実力はあるようですので彼女を採用するべきです。

> Setting aside her age, she seems to have real ability, so we should hire her.
