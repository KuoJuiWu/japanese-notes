# ~たが最後/たら最後

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | if... then absolutely... |

## Example

転勤になったら最後、最低でも3年は日本に帰ってこられない。

> Once I get transferred, I absolutely won't be able to return to Japan for at least three years.
