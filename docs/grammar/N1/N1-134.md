# ~にかこつけて

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | using as an excuse / under the pretext of... |

## Example

出張にかこつけて、各地の観光名所をまわるのが楽しみだ。

> I enjoy touring tourist attractions in various places using business trips as an excuse.
