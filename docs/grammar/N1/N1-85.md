# ~とあって

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | because of the special situation... |

## Example

超人気アイドルとあって、彼はすぐに大勢のファンに取り囲まれた。

> Because he is a super popular idol, he was immediately surrounded by a large crowd of fans.
