# ~と言えなくもない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | it could be said that... |

## Example

8割の確率で作ることができれば、成功と言えなくもない。

> If we can make it with an 80% success rate, it could be said to be a success.
