# ~に限ったことではない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | not limited to... |

## Example

高齢化社会は日本に限ったことではない。他の先進国も同じだ。

> An aging society is not limited to Japan. Other developed countries are the same.
