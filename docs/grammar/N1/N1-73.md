# ~でなくてなんだろう

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | if this isn't X, what is it? |

## Example

彼を見るとドキドキする。これが恋でなくてなんなんだろう。

> My heart pounds when I see him. If this isn't love, what is it?
