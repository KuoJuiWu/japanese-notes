# ~限りだ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | feel extremely... |

## Example

来月から息子が一人暮らしすることになった。さびしい限りだ。

> My son is going to live alone starting next month. I feel extremely lonely.
