# ~に言わせれば

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | if you ask... / according to... |

## Example

30代なんて、年寄りの私に言わせればまだまだ子供だ。

> People in their 30s, if you ask an old person like me, are still just children.
