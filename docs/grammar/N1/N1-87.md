# ~といい~といい

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | both... and... |

## Example

田中といい、鈴木といい、何で最近の新人はすぐに辞めてしまうのか。

> Both Tanaka and Suzuki... Why do recent new employees quit so quickly?
