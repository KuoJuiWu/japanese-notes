# ~を控えて

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in preparation for / facing... |

## Example

オリンピックの開幕を控えて、競技場の完成が急がれている。

> Facing the opening of the Olympics, the completion of the stadium is being rushed.
