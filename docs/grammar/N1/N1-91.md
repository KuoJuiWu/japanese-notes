# ~といえども

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | even if / even though... |

## Example

息子といえども、ここまで無能だと社長にさせるわけにはいかない。

> Even if he is my son, if he is this incompetent, I cannot let him become the president.
