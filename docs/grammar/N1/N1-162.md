# ~ばそれまでだ/たらそれまでだ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | that's the end of it / it's all over if... |

## Example

辛くても生きなさい。死んだらそれまでですよ。

> Live even if it's tough. If you die, that's the end of it.
