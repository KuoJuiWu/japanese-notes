# ~にしたところで/にしたって

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | even if... / even for... |

## Example

今さらタバコをやめるにしたところで、健康になるとは限らない。

> Even if you quit smoking now, it doesn't necessarily mean you'll become healthy.
