# ~なりとも

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | at least / even a little |

## Example

浮気したのは確かに悪いが、彼の気持ちも多少なりとも理解できる。

> Cheating is certainly bad, but I can understand his feelings at least a little.
