# ~べく

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in order to... |

## Example

金メダルを取るべく、毎日朝から晩まで練習ばかりしている。

> In order to win a gold medal, I do nothing but practice from morning to night every day.
