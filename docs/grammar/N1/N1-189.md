# ~ようによっては/ようでは

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | depending on how you look at it / depending on the method... |

## Example

彼に浮気されて別れたことも、考えようによってはよかったのかも。

> Breaking up with him because he cheated might have been a good thing, depending on how you look at it.
