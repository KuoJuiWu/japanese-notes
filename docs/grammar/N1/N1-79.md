# ~ても差し支えない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | it's okay to / doesn't matter if... |

## Example

新人が今日の会議に参加できなくても差し支えない。

> It doesn't matter if the new employee can't attend today's meeting.
