# ~さぞ/ さぞかし

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | must be / surely... |

## Example

今日は飲まず食わずだったとか。さぞお腹が空いているでしょう。

> I heard you haven't eaten or drunk anything today. You must be starving.
