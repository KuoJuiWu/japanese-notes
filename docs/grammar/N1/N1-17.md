# ~かれ〜かれ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | sooner or later / more or less |

## Example

毎日けんかしてるんだから、遅かれ早かれ離婚するでしょうね。

> Since they fight every day, they'll get divorced sooner or later.
