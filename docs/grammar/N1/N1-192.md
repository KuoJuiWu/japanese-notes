# ~を押して / を押し切って

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | overcoming / pushing past (opposition, illness)... |

## Example

周囲の反対を押し切って結婚したものの、二年で離婚した。

> Although they pushed past the opposition of those around them to marry, they divorced in two years.
