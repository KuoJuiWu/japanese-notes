# ~にもほどがある

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | there is a limit to... |

## Example

あれもこれも気に入らないなんて、わがままにもほどがある。

> Not liking this or that... there's a limit to how selfish you can be.
