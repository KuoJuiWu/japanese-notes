# ~羽目になる

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | end up in a bad situation... |

## Example

傘を電車に忘れてしまって、ぬれて帰る羽目になった。

> I forgot my umbrella on the train and ended up having to go home wet.
