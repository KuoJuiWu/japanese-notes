# ~に先駆けて

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | prior to / ahead of... |

## Example

この映画は、世界に先駆けて日本で上映されます。

> This movie will be screened in Japan ahead of the rest of the world.
