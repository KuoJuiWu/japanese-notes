# ~て済むことではない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | won't be resolved just by... |

## Example

人をけがさせておいて、謝って済むことではないのはわかってますか。

> Do you understand that injuring someone is not something that can be resolved just by apologizing?
