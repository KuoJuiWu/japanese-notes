# ~とはいえ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | although / even though... |

## Example

私はアメリカ人とはいえ日本で生まれ育ったので、日本語は問題ない。

> Although I am American, I was born and raised in Japan, so my Japanese is fine.
