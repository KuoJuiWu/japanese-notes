# ~故に/故の / 故だ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | because of / due to... |

## Example

お母さんはあなたを愛しているが故に、厳しく叱るんですよ。

> Your mother scolds you strictly precisely because she loves you.
