# ~というもの

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | for a long time / during this period... |

## Example

ここ一週間というもの、雨が降り続いて洪水が起きた地域もある。

> For the past week, it has kept raining, and there are even areas where floods have occurred.
