# ~ないまでも

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | even if not... at least... |

## Example

東京大学には合格しないまでも、どこかの国立大学になら受かるだろう。

> Even if he doesn't pass Tokyo University, he'll probably get into some national university.
