# ~きらいがある

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | have a tendency to (negative) |

## Example

父は、たとえ一大事でも一人で解決しようとするきらいがある。

> My father has a tendency to try and solve things on his own, even in a major crisis.
