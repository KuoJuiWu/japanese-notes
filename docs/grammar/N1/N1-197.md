# ~を機に

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | taking the opportunity of... |

## Example

病気になったのを機に、健康的な生活をするようになりました。

> Taking the opportunity of getting sick, I started living a healthy lifestyle.
