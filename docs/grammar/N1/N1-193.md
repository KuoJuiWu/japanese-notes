# ~を顧みず / も顧みず

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | without considering / disregarding... |

## Example

自分の危険も顧みず、子供を救うために川に飛び込んだ。

> Disregarding his own danger, he jumped into the river to save the child.
