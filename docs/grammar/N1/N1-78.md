# ~(で)もあり~(で)もある

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | is both... and... |

## Example

今日は私の誕生日でもあり、結婚記念日でもある。

> Today is both my birthday and my wedding anniversary.
