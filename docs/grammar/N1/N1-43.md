# ~ずには済まない / ないでは済まない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | must do / can't get away without doing... |

## Example

そんなに大きなミスでは、取引先までいって謝らないでは済まないだろう。

> With a mistake that big, you won't be able to get away without going to the client and apologizing.
