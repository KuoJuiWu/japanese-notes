# ~にあって

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in / during / under the condition of... |

## Example

社長という立場にあって、この時期に自分だけ休暇を取るわけにはいかない。

> In the position of company president, I cannot afford to take a vacation just for myself at this time.
