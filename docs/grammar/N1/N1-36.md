# ~さも

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | as if... |

## Example

彼はさも自分一人で作ったかのように話しているが、チームで作ったんだよ。

> He speaks as if he made it all by himself, but it was made by the team.
