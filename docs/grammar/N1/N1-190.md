# ~わ~わ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | what with... and... (usually negative pile-up) |

## Example

昨日は1日、甥と姪の面倒を見たが、騒ぐわ泣くわで大変だった。

> I took care of my nephew and niece all day yesterday, and what with the making noise and crying, it was exhausting.
