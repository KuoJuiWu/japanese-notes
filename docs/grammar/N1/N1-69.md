# ~て敵わない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | can't stand / unbearably... |

## Example

いくら赤ちゃんでも一晩中泣かれてはうるさくて敵わない。

> Even if it's a baby, I can't stand how loud it is being cried at all night long.
