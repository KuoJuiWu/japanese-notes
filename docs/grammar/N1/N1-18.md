# ~きっての

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | the best / the most in a particular group |

## Example

この学校きっての秀才の彼は、きっと東京の大学に行くんだろう。

> He is the brightest student in this school, so he will surely go to a university in Tokyo.
