# ~べくもない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | cannot possibly... |

## Example

幼いことから目立っていた彼がプロ選手になるのは、疑うべくもない。

> It is beyond doubt that he, who stood out from a young age, would become a pro player.
