# ~の極み

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | the ultimate / extreme of... |

## Example

高級ホテルで好きなように数日過ごし、ぜいたくの極みだった。

> Spending a few days at a luxury hotel doing whatever I wanted was the ultimate luxury.
