# ~は~てのことだ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | is because of... / thanks to... |

## Example

無事に第一希望の大学に合格できたのは、先生の教えがあってのことだ。

> Being able to safely pass my first-choice university is all thanks to my teacher's instruction.
