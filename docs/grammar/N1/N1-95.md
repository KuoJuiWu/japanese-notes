# ~といわず~といわず

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | not distinguishing between A and B / everywhere |

## Example

食事中といわず、寝る前といわず、どこでもスマホを離さない。

> Whether during meals or before bed, they never let go of their smartphone anywhere.
