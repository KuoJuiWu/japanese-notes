# ~てもどうにもならない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | it's no use even if... |

## Example

才能が全くなければ、いくら努力してもどうにもならない。

> If you have absolutely no talent, it's no use no matter how much effort you put in.
