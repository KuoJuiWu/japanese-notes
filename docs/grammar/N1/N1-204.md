# ~をものともせずに

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | without being deterred by... |

## Example

プレッシャーをものともせずに、余裕の演技で優勝した。

> Without being deterred by the pressure, they won with a composed performance.
