# ~ても知らない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | I don't care if / it's your own fault if... |

## Example

真面目に授業にも出ないて、卒業できなくても知らないよ。

> If you don't attend classes seriously and can't graduate, it's your own fault.
