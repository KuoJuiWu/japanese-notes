# ~あっての

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | thanks to / which owes everything to... |

## Example

私がこんなに日本語がうまくなったのも、先生の指導あってのことです。

> My Japanese has improved so much thanks entirely to my teacher's guidance.
