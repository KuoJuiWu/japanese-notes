# ~ともあろうものが

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | of all people / for someone in the position of... |

## Example

警察官ともあろうものが、犯罪者とグルだったなんて。

> To think that a police officer, of all people, was in cahoots with criminals.
