# ~に至って/至り

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | when it comes to / upon reaching the point of... |

## Example

商売を始めるに至って、お客様のことを第一に考えることが一番大切だ。

> Upon reaching the point of starting a business, putting the customer first is the most important thing.
