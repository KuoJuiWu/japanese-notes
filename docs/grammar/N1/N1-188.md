# ~ようにも〜ない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | even if one wanted to, one cannot... |

## Example

話を聞こうにも、泣いてちゃ聞けないよ。

> Even if I wanted to hear the story, I can't if you're crying.
