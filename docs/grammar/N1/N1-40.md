# ~ずじまい

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | ended up not doing... |

## Example

せっかく近くまで来たのに彼とは時間が合わず、会えずじまいだった。

> Even though I came all the way to his neighborhood, our schedules didn't match and I ended up not seeing him.
