# ~としてあるまじき

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | unworthy of / unbecoming for... |

## Example

患者を見捨てるなんて、医者としてあるまじき行為ですよ。

> Abandoning a patient is an act unworthy of a doctor.
