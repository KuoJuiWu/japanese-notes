# なんという / なんと / なんて

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | how / what a... |

## Example

あれほどの地震で死者が一人もでなかったとは、なんという奇跡。

> To think that not a single person died in such a massive earthquake, what a miracle!
