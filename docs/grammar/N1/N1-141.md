# ~に即して / に即した

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in line with / according to... |

## Example

時代に即して考えを変えないと、若者の理解は得られません。

> If we don't change our thinking in line with the times, we won't gain the understanding of young people.
