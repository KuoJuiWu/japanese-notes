# ~にかかっては/ にかかったら / にかかると / かかれば

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | if handled by / if it comes to... |

## Example

どんな肉でも、一流シェフの手にかかっては柔らかくておいしい肉に変わる。

> Any kind of meat will turn into tender and delicious meat if handled by a first-class chef.
