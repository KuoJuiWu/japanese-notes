# ~いかんにかかわらず / いかんによらず / いかんを問わず

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | regardless of... |

## Example

成績いかんにかかわらず、出席日数さえあれば卒業できる。

> Regardless of your grades, you can graduate as long as you have the required attendance.
