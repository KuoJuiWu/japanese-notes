# であれ/であろうと

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | even if / no matter what |

## Example

どんな理由であれ、法律を守らなかったらつかまるんだよ。

> No matter the reason, you will get caught if you don't follow the law.
