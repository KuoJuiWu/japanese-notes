# ~こそ〜が/けれど

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | although X, Y is true |

## Example

部長は話し方こそ怖いが、実はとてもいい人なんです。

> Although the manager's way of speaking is scary, he is actually a very nice person.
