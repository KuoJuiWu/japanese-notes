# ~ともなれば/ともなると

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | when it comes to... / once it becomes... |

## Example

年末年始ともなると、実家に帰省する人が多く新幹線の切符も取りにくい。

> When the New Year holidays arrive, many people return to their hometowns, making it hard to get Shinkansen tickets.
