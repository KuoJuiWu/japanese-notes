# ~うちに入らない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | doesn't count as / doesn't fall under... |

## Example

お菓子なんて食事のうちに入らないだろう。ちゃんと食べなきゃ。

> Snacks don't count as a meal. You have to eat properly.
