# ~に至っては

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | when it comes to... (extreme case) |

## Example

こんな日本語、日本人でさえ難しいのに、外国人に至っては全く理解できないだろう。

> This kind of Japanese is difficult even for Japanese people, so when it comes to foreigners, they probably won't understand it at all.
