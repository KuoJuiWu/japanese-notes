# ~ずくめ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | entirely / completely covered in... |

## Example

防犯カメラに写っているのは黒ずくめの男だった。

> The person caught on the security camera was a man dressed entirely in black.
