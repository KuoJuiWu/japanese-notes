# ~かと思いきや

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | contrary to expectations... |

## Example

田中さんは離婚したかと思いきや、数ヶ月で再婚した。

> I thought Tanaka-san had gotten divorced, but contrary to my expectations, he remarried a few months later.
