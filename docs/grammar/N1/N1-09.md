# ~かたがた

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | while at it / for the purpose of... |

## Example

この間のお礼かたがたお伺いしたいんですが、よろしいでしょうか。

> I'd like to visit you to express my thanks for the other day, if that's alright.
