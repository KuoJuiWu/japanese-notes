# ~も同然だ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | virtually the same as / practically... |

## Example

相手チームのエースがけが？じゃあ、今日の試合は勝ったも同然だな。

> The opposing team's ace is injured? Then today's match is practically won.
