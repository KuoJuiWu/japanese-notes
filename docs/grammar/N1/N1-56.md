# ~だの〜だの

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | things like... and... |

## Example

アイスだの、ケーキだの、そんなのばっかり食べてるから太るんだ。

> You gain weight because you only eat things like ice cream and cake.
