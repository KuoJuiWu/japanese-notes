# ~放題

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | as much as one wants / leaving unchecked |

## Example

ケーキ食べ放題プラン、一人二時間で3000円です。

> The all-you-can-eat cake plan is 3000 yen per person for two hours.
