# ~までだ/までのことだ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | only / just... (no other reason) |

## Example

私の意見を正直に言ったまでです。文句を言われても困ります。

> I merely stated my opinion honestly. I'd be troubled if you complain about it.
