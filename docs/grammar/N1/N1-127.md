# ~に値する

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | worthy of... |

## Example

自分勝手な彼がとうとう会社をクビになったが、全く同情に値しない。

> Selfish as he is, he finally got fired from the company, but he is completely unworthy of sympathy.
