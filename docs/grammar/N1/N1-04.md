# いずれにしても / いずれにしろ / いずれにせよ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | in any case / either way |

## Example

タクシーで行こうが、バスで行こうが、いずれにしろもう遅刻です。

> Whether we take a taxi or a bus, we're going to be late either way.
