# ~としたことが

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | of all people / for someone like... |

## Example

私としたことが、こんな簡単な問題を間違えるなんて。

> To think that someone like me would make a mistake on such a simple problem.
