# ~を皮切りに

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | starting with... |

## Example

渋谷のこの店を皮切りに、全国に店を出していく予定だ。

> Starting with this store in Shibuya, we plan to open stores nationwide.
