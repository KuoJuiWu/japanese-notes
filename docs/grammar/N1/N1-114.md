# ~ないとも限らない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | there is a possibility that... / not completely impossible |

## Example

この地域にも大地震が起こらないとも限らないので、準備はしておいたほうがいい。

> Since it's not impossible for a major earthquake to happen in this area too, it's better to be prepared.
