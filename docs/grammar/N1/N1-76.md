# ~手前

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | considering / out of consideration for... |

## Example

絶対に勝つと言った手前、負けるわけにはいかない。

> Considering I said I would absolutely win, I cannot afford to lose.
