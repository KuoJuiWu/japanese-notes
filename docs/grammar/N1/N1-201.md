# ~を踏まえて

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | based on / taking into account... |

## Example

今回の試験結果を踏まえて、目指す大学のレベルを少し落とした。

> Based on the results of this exam, I slightly lowered the level of the university I'm aiming for.
