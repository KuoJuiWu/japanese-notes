# ~ぶる

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | act like / pretend to be... |

## Example

彼女はいい人ぶっているが、陰で人の悪口ばかり言ってるよ。

> She pretends to be a good person, but she just speaks ill of people behind their backs.
