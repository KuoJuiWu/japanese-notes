# ~にも増して

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | more than ever / even more than... |

## Example

今年の夏は例年にも増して暑い。熱中症にならないように気をつけて。

> This summer is even hotter than usual years. Be careful not to get heatstroke.
