# ~とまではいかないが / とまではいかないにしても

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | not to the extent of... but... |

## Example

満点とまではいかないが、それに近い点数で合格した。

> I didn't get a perfect score, but I passed with a score close to it.
