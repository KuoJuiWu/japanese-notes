# ~をよそに

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | indifferent to / disregarding... |

## Example

周囲の心配をよそに、彼は自分の信じる道を突き進んでいる。

> Disregarding the worries of those around him, he pushes forward on the path he believes in.
