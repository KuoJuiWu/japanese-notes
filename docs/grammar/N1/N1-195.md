# ~をかねて

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | also for the purpose of / combined with... |

## Example

親孝行をかねて、家族みんなで温泉旅行にでも行くつもりだ。

> I plan to go on a hot spring trip with the whole family, also to show filial piety.
