# ~ごとく

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | like / as if... |

## Example

真夏に外で作業すると、汗が滝のごとく流れ落ちる。

> When working outside in midsummer, sweat pours down like a waterfall.
