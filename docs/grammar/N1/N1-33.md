# ~こともあって

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | partly because... |

## Example

久しぶりに会ったこともあって、話しても話し足りなかった。

> Partly because we hadn't seen each other in a long time, we couldn't talk enough.
