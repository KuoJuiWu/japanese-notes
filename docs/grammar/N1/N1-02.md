# ~いかんだ / いかんでは / いかんによっては

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | depending on... |

## Example

面接の結果いかんでは、アメリカに引っ越すことになるかもしれません。

> Depending on the results of the interview, I might have to move to America.
