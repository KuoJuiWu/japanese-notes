# ~のをいいことに

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | taking advantage of... |

## Example

彼は誰も見ていないのをいいことに、仕事中ずっと寝ていたんです。

> Taking advantage of the fact that no one was watching, he slept throughout work.
