# ~のも無理はない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | it's no wonder that... |

## Example

デートで浮気相手とはちあわせ、そりゃ彼女にふられるのも無理はない。

> Running into his cheating partner on a date... it's no wonder he got dumped by his girlfriend.
