# ~始末だ

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | end up doing (negative result) |

## Example

遅刻したと思ったら、仕事中に居眠りまでする始末だ。

> Just when I thought him being late was bad enough, he even ends up falling asleep during work.
