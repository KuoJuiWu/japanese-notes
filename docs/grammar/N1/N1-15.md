# ~がましい

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | sounds like / looks like (negative nuance) |

## Example

一年前に別れた彼女に連絡するなんて、未練がましいよ。

> Contacting a girlfriend you broke up with a year ago looks like you're clinging to the past.
