# やれ〜やれ〜

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | what with... and... |

## Example

やれ塾だ、やれスポーツクラブだと、最近の子供は大人より忙しい。

> What with cram school and sports clubs, children these days are busier than adults.
