# ~までもない/までもなく

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | no need to / goes without saying... |

## Example

このくらいの風邪なら、薬を飲むまでもないですよ。

> For a cold of this level, there's no need to take medicine.
