# ~とは比べものにならない

| | |
|---|---|
| **Level** | N1 |
| **Meaning** | cannot be compared to / is nothing compared to... |

## Example

兄は、弟の僕とは比べものにならないほど頭がいい。

> My older brother is so smart that I, his younger brother, cannot even be compared to him.
