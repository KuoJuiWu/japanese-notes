# こ/そ/あ/ど

| | |
|---|---|
| **Level** | N5 |
| **Meaning** | demonstrative pronouns |

## Example

こ: 近い、そ: 相手に近い、あ: 遠い、ど: わからない

> こ: near speaker, そ: near listener, あ: far from both, ど: unseen/unknown
