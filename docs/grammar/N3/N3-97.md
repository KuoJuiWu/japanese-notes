# ~んだって

| | |
|---|---|
| **Level** | N3 |
| **Meaning** | I hear that... (colloquial) |

## Example

スミスさん、卒業したら国に帰るんだって。

> I heard that Mr. Smith is going to go back to his country after he graduates from school.
