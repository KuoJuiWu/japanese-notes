# ~はもちろん〜も

| | |
|---|---|
| **Level** | N3 |
| **Meaning** | Not to mention... / let alone... |

## Example

私は自転車はもちろん、バイクにも乗れる。

> I can ride a motorbike, to say nothing of a bicycle.
