# ~つもりだった

| | |
|---|---|
| **Level** | N3 |
| **Meaning** | I had planned to... (but didn't) |

## Example

アイスは食べないつもりだったけど、食べちゃった

> I was not going to eat ice cream but I ended up eating it.
