# もし〜たなら

| | |
|---|---|
| **Level** | N3 |
| **Meaning** | If (hypothetical past) |

## Example

もし日本に来なかったなら、日本語を勉強していなかったと思う。

> If I hadn't come to Japan, I would not have studied Japanese.
