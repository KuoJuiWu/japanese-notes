# ~ことに

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | To my (surprise, disappointment, etc.)... |

## Example

残念なことに、いちばん会いたかった佐藤さんが来られなくなった。

> Unfortunately, the person I wanted to see the most, Sato-san, could not come.
