# ~かねない

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Might (bad consequence)... |

## Example

ちゃんと病院に行かないと、病気が悪化しかねないよ

> If you don't go to the hospital properly, your illness could get worse.
