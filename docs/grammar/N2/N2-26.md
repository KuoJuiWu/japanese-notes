# ~となると

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | When it comes to... / If it comes to that... |

## Example

海外に住むとなると、友だちにも簡単に会えなくなるなあ。

> If I were to live abroad, I would not be able to see my friends as often.
