# ~に相違ない

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Without a doubt... |

## Example

この学校で成績トップの彼女なら、東大に合格するに相違ない

> She is the student with the highest grades at this school, so she will definitely get into Tokyo University.
