# 〜くらいなら

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Rather than (do X, I would do Y) |

## Example

あの人に謝るくらいなら、殴られたほうがいい。

> I'd rather he/she punch me than I apologize to him/her.
