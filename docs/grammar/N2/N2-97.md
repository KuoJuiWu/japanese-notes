# ~かいあって/~かいもなく

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Thanks to the effort of... / despite the effort of... |

## Example

毎日のトレーニングのかいあって、優勝することができた

> The daily training was worth it, and I was able to win.
