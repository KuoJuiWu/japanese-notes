# ~というか〜というか

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Or rather... / I'm not sure if it's A or B... |

## Example

彼女のドレスは、派手というか奇抜というか、とても目立っていた

> Her dress was standing out, not to say flashy or outlandish.
