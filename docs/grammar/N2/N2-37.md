# ~ざるをえない

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Have no choice but to... |

## Example

父に言われたなら、嫌でも弟の面倒を見ざるをえない。

> My father told me to do it, so even if I didn't want to, I'd have to take care of my younger brother.
