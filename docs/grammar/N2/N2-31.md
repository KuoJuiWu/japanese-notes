# ~どころか

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Far from... |

## Example

先日の旅行は、楽しいどころかハプニングばかりで本当に疲れた。

> The day's trip was not fun at all, despite my expectations, and I was really tired from all the unexpected events.
