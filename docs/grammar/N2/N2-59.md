# ~どころではない

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Not the time/place for... |

## Example

子どもが大けがをして、誕生日パーティーどころじゃなかった。

> My child was seriously injured and we couldn't have his birthday party.
