# ~に反して

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Contrary to... |

## Example

親の期待に反して、息子は医者にはならず、小さな店をしている。

> Contrary to our expectations, my son did not become a doctor and runs a small store.
