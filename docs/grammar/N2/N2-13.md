# ~ものだから

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Because... (excuse/reason) |

## Example

昨日は忙しかったものだから、この仕事はまだ終わっていないんです。

> I was busy yesterday, so I haven't finished this job yet.
