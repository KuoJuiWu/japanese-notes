# ~つつ(も)

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Even though / despite (doing) |

## Example

やめようと思いつつ、ついお酒を飲みすぎてしまう

> I try to stop, but I end up drinking too much.
