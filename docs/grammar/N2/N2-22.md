# ~ものなら/~もんなら

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | If it were possible... |

## Example

娘が病気でつらそうだ。代われるものなら、代わってあげたい。

> My daughter is sick and in pain. If it were possible, I would like to switch her body and mine.
