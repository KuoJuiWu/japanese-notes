# ~か~ないかのうちに

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Right after / barely... |

## Example

店が開くか開かないかのうちに、人気商品は全部売り切れた

> As soon as the store opened, all the popular items were sold out.
