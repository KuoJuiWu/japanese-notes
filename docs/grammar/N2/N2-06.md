# ~からといって/からって

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Just because... |

## Example

日本に住んでいるからといって、日本語が話せるようになるわけではない。

> Just because you live in Japan does not mean you will be able to speak Japanese.
