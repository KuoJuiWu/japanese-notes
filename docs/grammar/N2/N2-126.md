# ~とは

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | The definition of X is... |

## Example

土足厳禁とは、靴をはいたまま中に入ってはいけないという

> “土足厳禁” means that you should not go inside with your shoes on.
