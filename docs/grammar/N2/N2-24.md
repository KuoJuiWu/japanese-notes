# ~さえ〜ば

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | As long as / if only... |

## Example

スマホさえあれば、現金やカードがなくても買い物ができる。

> With just a smartphone, you can make purchases without cash or credit cards.
