# ~(より)ほかない

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Have no choice but to... |

## Example

こんな田舎にタクシーなんて来ないし、歩いて帰るほかないですね。

> It's a very rural area, so cabs won't come, and we have to walk home.
