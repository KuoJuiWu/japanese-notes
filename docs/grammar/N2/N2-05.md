# ~にかかわらず/にかかわりなく

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Regardless of... |

## Example

年れいにかかわらず、優秀なら合格しますよ。

> Regardless of your age, if you are good enough, you will be accepted.
