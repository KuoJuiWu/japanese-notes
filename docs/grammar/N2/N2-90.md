# ~あげく

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | In the end / finally (after a long process)... |

## Example

3時間も待ったあげく、売り切れで買えなかった。

> After waiting for three hours, I couldn't buy it because it was sold out.
