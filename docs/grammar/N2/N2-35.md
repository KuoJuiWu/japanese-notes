# ~だけまし

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | At least / it's better than... |

## Example

ハイキングには寒いけど、雪が降らないだけましですね。

> The weather is cold for hiking, but it's still good because it's not snowing.
