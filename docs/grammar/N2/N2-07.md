# ~にしても/にしろ/にせよ

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Even if... |

## Example

忙しかったにしても、電話ぐらいはできたでしょう?

> I understand that you were busy, but you could have at least made a phone call, couldn't you?
