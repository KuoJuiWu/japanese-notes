# ~ようものなら

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | If one were to do... (leads to bad result) |

## Example

妻の誕生日を忘れようものなら、家に入れてもらえないかもしれない。

> If I were to forget my wife's birthday, she might not let me in the house.
