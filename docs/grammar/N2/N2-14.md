# ~のことだから

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Since it is... (expectations based on a person's nature) |

## Example

(いつも遅刻する)スミスさんのことだから、今日も遅れてくるんじゃない?

> Since Smith-san is always late, he would probably be late again today.
