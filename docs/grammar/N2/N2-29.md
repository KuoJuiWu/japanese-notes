# ~のもと(で)

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Under (supervision/influence)... |

## Example

最新のロボットを作りたいなら、この大学の田中先生のもとで学ぶべきだよ。

> If you want to build the latest robot, you should study under Professor Tanaka of this university.
