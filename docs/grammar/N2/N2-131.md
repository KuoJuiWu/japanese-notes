# ~ものがある

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | There is a feeling of... |

## Example

いくらかわいくても、毎日孫のめんどうを見るのは、正直つらいものがある

> To be honest, it's hard to take care of my grandchildren every day, no matter how cute they are.
