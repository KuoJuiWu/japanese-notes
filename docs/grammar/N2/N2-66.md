# ~はもとより(~も)

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | Not only... but of course... |

## Example

あの歌手は、歌はもとより、表現力もすばらしい

> That singer is not only a great singer, but also a great expressive artist.
