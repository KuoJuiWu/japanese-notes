# ~だけのことはある

| | |
|---|---|
| **Level** | N2 |
| **Meaning** | No wonder / it shows... |

## Example

彼だけは全然疲れていない。やっぱり若いだけのことはある

> Only he is not tired at all. It truly shows the advantage of being young.
