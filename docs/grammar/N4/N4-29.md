# こ/そ/あ②

| | |
|---|---|
| **Level** | N4 |
| **Meaning** | (to refer to things not visible to the speaker or the hearer) |

## Example

ねえ、この話、知ってる? 山田さんがけっこん!

> Do you know this story? Mr. Yamada is getting married.
