# ~てから VS ~あとで

| | |
|---|---|
| **Level** | N4 |
| **Meaning** | Continuous action vs Completed action |

## Example

ごはんを食べてから、薬を飲む。 VS ごはんを食べたあとで、薬を飲む。

> (You don't know if you're done eating... The action continues.) VS (The "eating of meal" has to be completed.)
