# ~し〜し

| | |
|---|---|
| **Level** | N4 |
| **Meaning** | (expressing multiple reasons or repeating similar things) |

## Example

ラーメンはかんたんにできるし安いし、おいしい。

> Ramen is easy to cook, cheap and delicious.
