# ~ようだ/みたいだ

| | |
|---|---|
| **Level** | N4 |
| **Meaning** | it seems that... |

## Example

せきも出るし、頭も痛いし、かぜをひいたようです

> I've got a cough, and my head aches; it seems I might have caught a cold.
