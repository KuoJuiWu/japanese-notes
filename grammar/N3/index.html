    N3 — Intermediate Grammar Patterns¶

    Pattern Meaning
    ~おかげで Thanks to...
    ~せいで Because of... / Due to...
    ~からこそ Because...
    ~によって① Due to...
    ~(んだ)もの/もん Explanation, reason
    ~ことはない There is no need to...
    ~べきだ Should...
    ~ように So that / in order to (or as a soft command)
    ~こと Rule or command
    ~によって② By means of...
    ~によれば/~によると According to...
    ~を通して/通じて① Through / via
    ~から〜にかけて From... through...
    ~を中心に Focused on / centered on
    ~を通して/通じて② Throughout (a period)
    ~出す Suddenly begin to...
    縮約形(Contraction) Colloquial contractions (e.g. ~ておく -> ~とく)
    ~ないと/なきゃ/なくちゃ Must / have to
    ~たって Even if... (colloquial)
    〜っけ Trying to recall something
    ~と、〜た/~たら、〜た When... (discovery/surprise)
    ~としたら/~とすれば If (hypothetical)
    ~ば~のに If only... (regret)
    もし〜たなら If (hypothetical past)
    ~がち Tend to...
    ~だらけ Covered with / full of (usually negative)
    〜っぱなし Leaving something in an improper state
    〜っぽい -ish / looking like...
    ~気味 Slight tendency / feeling a bit...
    ~ふりをする Pretend to...
    ~まま As it is / keeping the same state
    ~みたい Looks like / seems like
    (まるで)~ようだ Just like...
    ~がる Shows signs of (third person's feelings)
    ~( )きり Only / just (indicates a limit)
    ~ばかり Nothing but / always
    ~てほしい/~てもらいたい I want someone to do...
    ~と/たら/ば、いい(なあ) I wish / I hope...
    ~ますように/~ませんように I pray/hope that...
    ~として As... (role/capacity)
    ~にとって For / to (someone's perspective)
    ~ばかりか/~ばかりでなく Not only... but also...
    ~はもちろん〜も Not to mention... / let alone...
    たとえ〜ても Even if...
    どんなに〜ても/いくら〜ても No matter how much...
    ~つもりだった I had planned to... (but didn't)
    ~わけだ No wonder... / That explains why...
    つまり In other words
    ~ということだ① It means that...
    ~ことがある/~こともある There are times when...
    に対して① Towards / against...
    向け Targeted for / aimed at...
    ~というと/~といえば/~といったら Speaking of...
    ~について About / regarding...
    ~ことだ Should / must (giving advice)
    ~上げる/~上がる To finish doing...
    ~切る/~切れる To do completely / entirely
    ~たところ When I did... (result)
    ~うちに① While (doing something, another thing happens)
    ~かけ Half-done / in the middle of...
    ~てからでないと/~てからでなければ Unless... / Not until...
    ~てはじめて Not until... (did I realize)
    ~ところだった Almost did / was just about to...
    ~たとたん As soon as...
    ~とともに Together with / As... then...
    ~たて Just done / freshly done
    ~ついでに While doing..., might as well...
    ~始める Start to...
    ~終わる Finish doing...
    ~ようとする Be about to / try to...
    ~ようとしない Doesn't try to / shows no sign of...
    ~うちに② While (a condition holds true)
    ~において In / at (place, time, context)
    ~最中に Right in the middle of...
    ~たものだ Used to (do something in the past)
    ~ないことはない It's not that I don't...
    ~ばよかった Should have...
    ~ずにはいられない Can't help but...
    ~てしょうがない/~てしかたがない Can't help / unbearably...
    ~わけにはいかない Cannot afford to / must not...
    ~たびに Every time...
    ~とおり(に) Just as / according to...
    ~にしたがって As (one thing happens, another happens)
    ~ていく/~てくる Will become / continue to...
    ~続ける Continue doing...
    ば~ほど The more... the more...
    ~くせに Even though / despite (blaming/criticizing)
    ~くらい、ぐらい/~ほど To the extent that...
    ~こそ For sure / definitely (emphasis)
    ~(で)さえ Even...
    ~なんか/~なんて Things like... (belittling/modesty)
    ~まで Even / as far as...
    ~という Called / named...
    ~ということだ② It is said that / I heard...
    [命令形/禁止形]と言われる To be told to / not to...
    らしい Seems like / typical of...
    ~んだって I hear that... (colloquial)
    ~かなあ I wonder...
    もしかすると/もしかしたら〜かもしれない Perhaps / maybe...
    ~に違いない Must be / without a doubt...
    ~しかない Have no choice but to...
    じゃない Isn't it? (seeking agreement)
    ~ずに Without doing...
    ~わけではない It doesn't mean that...
    ~ことは〜が It is true that... but...
    まったく/めったに/決して/少しも〜ない Not at all / rarely / never
    ~とは限らない Not necessarily so...
    ~かわりに Instead of...
    ~というより Rather than...
    ~にかわって/~にかわり In place of / on behalf of...
    ~にしては Considering it's / for...
    ~に対して② In contrast to...
    ~に比べて Compared to...
    ~ほど･･･は･･･ない/~くらい･･･は･･･ない There is nothing as... as...
    ~によって③ Depending on...
    ~わりに(は) Although / despite / relatively...
    ~というのは The meaning of X is...
    ~といっても Even if I say that...
    ~に関して/関する Regarding / related to...
